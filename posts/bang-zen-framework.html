Bang: Minimalist JavaScript View Framework

By Cris, Undated

Modern web UI frameworks often burden developers with virtual DOM overhead and complex build steps, stifling the simplicity of crafting interactive interfaces. I sought a lighter path with ! (Bang), a minimalist JavaScript view framework that leverages Custom Elements for component-based design, introduces a novel HTML comment-based syntax for void tags like <!sg-circle />, and manages state-driven interactivity without a virtual DOM. In the web’s Zen garden, where every pebble counts, Bang is a raked path of clarity, inviting developers to create with effortless precision.

Bang uses Shadow DOM for scoped styles, ensuring components remain isolated, and relies on native JavaScript templating for rendering. The framework’s state management, via functions like setState, drives dynamic updates, while event handlers bring interactivity without build-time complexity. In a drawing app example, Bang powers a canvas where users add and resize circles, each circle a Custom Element with its own state and behavior, as serene as a garden’s still pond.

class CircleDrawer extends Base {
  static Radius = 32;

  AddCircle(clickEvent) {
    if (clickEvent.target.matches('sg-circle')) return;
    const {clientX, clientY} = clickEvent;
    const frame = clickEvent.target.closest('.canvas');
    const {left, top} = frame.getBoundingClientRect();
    const r = CircleDrawer.Radius;
    const x = clientX - left;
    const y = clientY - top;
    const state = cloneState('circleDrawer');
    state.circles.push({
      key: `${x},${y}`,
      x, y, 
      radius: r
    });
    setState('circleDrawer', state, {save: true});
  }

  OpenSizer(event) {
    if (event.type === 'contextmenu') {
      event.preventDefault();
    }
    const selected = event.target.dataset.key;
    const state = cloneState('circleDrawer');
    state.selected = selected;
    setState('circleDrawer', state);
  }
}

This code defines a CircleDrawer component that adds circles to a canvas and opens a sizing modal, using Bang’s state management and event handling. The innovation lies in its restraint: zero virtual DOM, native templating with Custom Elements, and a clever void tag syntax (<!sg-circle />) that simplifies component usage, all while maintaining interactivity as naturally as a Zen master rakes sand.

Building Bang was like tending a Zen garden, each line of code a deliberate stroke toward serenity. At DOSAYGO, we value clarity and simplicity, crafting tools that empower without overwhelming.
